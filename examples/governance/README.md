# Governance module examples

Governance module functionality demonstration using published akash SDK.

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python governance.py
```

## Features

▫️ Proposal queries and filtering
▫️ Voting functionality  
▫️ Deposit management
▫️ Parameter change proposals
▫️ Text proposals

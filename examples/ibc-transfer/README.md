# IBC token transfer e2e tests

Copy of IBC token transfer e2e tests using published akash SDK.

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python ibc_transfer.py
```

▫️ Wallet balance verification
▫️ Channel discovery and structure
▫️ Transfer message construction
▫️ Transfer simulation

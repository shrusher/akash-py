# Slashing module examples

Slashing module functionality demonstration using published akash SDK.

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python slashing.py
```

## Features

▫️ Validator signing information
▫️ Slashing parameters
▫️ Unjail transactions
▫️ Validator monitoring
▫️ Testnet and mainnet support

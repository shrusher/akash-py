# Wallet module examples

Wallet module functionality demonstration using published akash SDK.

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python wallet.py
```

## Features

▫️ Mnemonic generation and validation
▫️ Address derivation from mnemonic
▫️ Private key management
▫️ Transaction signing
▫️ Key recovery and validation

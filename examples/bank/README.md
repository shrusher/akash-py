# Bank module examples

Bank module functionality demonstration using published akash SDK.

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python bank.py
```

## Features

▫️ Balance queries
▫️ Account information
▫️ Address validation
▫️ Supply queries
▫️ Fee estimation
▫️ Send transactions (testnet only)

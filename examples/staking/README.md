# Staking module examples

Staking module functionality demonstration using published akash SDK.

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python staking.py
```

## Features

▫️ Validator queries and filtering
▫️ Delegation management
▫️ Unbonding and redelegation
▫️ Validator creation and editing
▫️ Staking parameters and pool info

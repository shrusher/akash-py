# Market module examples

Market module functionality demonstration using published akash SDK.

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python market.py
```

## Features

▫️ Order management
▫️ Bid creation and closure
▫️ Lease management
▫️ Market state filtering
▫️ Testnet and mainnet support
